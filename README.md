# petraBT.github.io

Landing page for Petra Bonfert-Taylor's interactive textbooks, used in ENGS 20
("Introduction to Scientific Computing") and ENGS 28 ("Embedded Systems
Programming") at Thayer School of Engineering, Dartmouth College.

Live site: [engs20book.thayer.dartmouth.edu](https://engs20book.thayer.dartmouth.edu/)

## Projects

- [Introduction to C Programming](https://petrabt.github.io/C-Programming) — live version, plus a [static/offline version](https://petrabt.github.io/C-Programming/static/)
- [Introduction to Matlab Programming](https://petrabt.github.io/Matlab/)
- [ENGS 28 C Programming Guide](https://petrabt.github.io/ENGS28Guide)

## About this repo

This repo holds only the landing page (`index.html`) for the site above. Each
project linked from it lives in its own separate repository.
